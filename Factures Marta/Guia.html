Posar l'app en línia, pas a pas

Tres parts: 1) Firebase (les teves dades al núvol + login) · 2) GitHub (publicar l'app) · 3) instal·lar-la a l'iPhone. Uns 15 minuts en total.

Part 1

Firebase — dades al núvol i login

  1. Entra a console.firebase.google.com amb el teu compte de Google i clica Crear un projecte. Posa-li un nom (p. ex. factures-marta), accepta i continua. Pots desactivar Google Analytics, no cal.
  2. Dins el projecte, clica la icona web </> ("Add app" → Web). Posa-li un sobrenom i clica Registrar app. Et mostrarà un bloc const firebaseConfig = {…} amb 6 línies.
  3. Copia aquests 6 valors (apiKey, authDomain, projectId, storageBucket, messagingSenderId, appId) i enganxa'ls al fitxer firebase-config.js, substituint els textos PEGA_….
  4. Al menú lateral: Build → Authentication → Get started. A la pestanya Sign-in method, activa Email/Password i desa.
  5. A la pestanya Users, clica Add user. Posa el teu correu i una contrasenya: aquest serà el teu únic usuari (el que faràs servir per entrar a l'app).
  6. Al menú: Build → Firestore Database → Create database. Tria Production mode i una ubicació europea (p. ex. eur3). Crea-la.
  7. A la pestanya Rules de Firestore, enganxa exactament això i clica Publish:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{uid} {
      allow read, write: if request.auth != null
                         && request.auth.uid == uid;
    }
  }
}
🔒
És segur? Sí. Les 6 línies de firebase-config.js no són secretes (van sempre al navegador). El que protegeix les teves dades és el login + aquestes Rules: només el teu usuari pot llegir o escriure les teves dades.

Part 2

GitHub — publicar l'app gratis

  1. Crea un compte a github.com (si no en tens). Després clica New repository, posa-li un nom (p. ex. factures), deixa'l Public i crea'l.
  2. Puja tots els fitxers del projecte (botó Descarregar projecte de baix → descomprimeix → arrossega-ho tot a la pàgina del repo a Add file → Upload files). Important: inclou les carpetes _ds i assets senceres.
  3. Al repo: Settings → Pages. A Source tria Deploy from a branch, branca main, carpeta / (root), i Save.
  4. Espera 1–2 minuts. La teva app quedarà a:
    https://EL-TEU-USUARI.github.io/factures/app.html
💡
Cada cop que vulguis canviar alguna cosa, torna a pujar els fitxers modificats. Si edites firebase-config.js, l'app es reconnecta sola.

Part 3

Instal·lar-la a l'iPhone

  1. Obre la URL …github.io/factures/app.html amb Safari (ha de ser Safari, no Chrome).
  2. Toca el botó Compartir ↑ i tria Afegir a la pantalla d'inici.
  3. Obre l'app des de la nova icona. Et demanarà el login (el correu i contrasenya del pas 1.5).
  4. Repeteix al PC obrint la mateixa URL i fent login: veuràs exactament les mateixes dades, sincronitzades a l'instant.

Bo per saber

• Mentre no enganxis la config de Firebase, l'app funciona en mode local (només en aquest dispositiu) — així mai queda bloquejada.
• El botó Còpia de seguretat segueix funcionant com a còpia extra i per endur-te les dades.
• Per tancar la sessió: obre Còpia de seguretat → Tancar sessió.
• Funciona sense connexió: els canvis es guarden i es sincronitzen quan tornis a tenir internet.
